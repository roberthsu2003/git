## 主要分支（master)
> 一個專案至少要有一個分支
> 
> 預設分支的名稱是master
> 
> master分支是一個指向單一commit的指標,可以想像是一張貼在commit上的標籤
> 
> master是可以移動的標籤

## HEAD是什麼？

> HEAD代表目前在哪一個commit上
> 
> HEAD也是可以移動的標籤
> 
> HEAD一般都貼在分支上,當分支移動時,HEAD也會跟著移動
>
> HEAD可以切換到不同分支,HEAD一次只可以貼在一個分支上
